An Evening In was created with Inform and has IFID 90E28CDA-CB9B-4E54-B607-70B7C0A34897. To play a work like this one, you need an interpreter program: many are available, among them Zoom for Mac OS X and for Unix; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
